# 图片转PDF工具

一个简单易用的在线工具，可以将多张图片合并转换为PDF文档。

## 功能特点

- 支持拖拽上传图片
- 支持多图片批量上传
- 支持jpg、png、gif等常见图片格式
- 实时预览上传的图片
- 可删除已上传的图片
- 自动调整图片尺寸以适应A4纸张
- 生成的PDF文档每张图片独占一页

## 使用方法

1. 打开网页工具
2. 点击上传区域或将图片拖拽到上传区域
3. 查看预览区域中的图片，可根据需要删除不需要的图片
4. 点击"生成PDF"按钮，即可下载合成的PDF文件

## 技术栈

- HTML5
- CSS3
- JavaScript
- jsPDF库
